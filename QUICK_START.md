## 永久解决方案：配置DNS

### 步骤1：添加DNS记录

在你的域名注册商（如阿里云、腾讯云等）添加：

```
类型: A
主机记录: @
记录值: 117.72.39.169

类型: A
主机记录: www
记录值: 117.72.39.169
```

### 步骤2：等待DNS生效

通常需要10分钟到24小时。

验证方法：
```bash
dig hao246.cn
# 应该返回: 117.72.39.169
```

### 步骤3：获取正式SSL证书

DNS生效后，在服务器上运行：

```bash
ssh root@117.72.39.169

# 停止Nginx
systemctl stop nginx

# 获取证书
certbot certonly --standalone \
  -d hao246.cn \
  -d www.hao246.cn \
  --email admin@hao246.cn \
  --agree-tos \
  --non-interactive

# 启动Nginx
systemctl start nginx
```
