### 🎁 第二步：运行安装脚本

**Windows 用户：**

1. 下载 `install_claude_code.bat` 文件
2. 右键点击文件，选择"以管理员身份运行"
3. 按照提示完成配置

**Mac/Linux 用户：**

1. 下载 `install_claude_code.sh` 文件
2. 打开终端，进入文件所在目录
3. 运行脚本：
```bash
chmod +x install_claude_code.sh
./install_claude_code.sh
```

**脚本会自动完成：**
- ✅ 检查 Node.js 和 npm 环境
- ✅ 自动升级 npm 到最新版本
- ✅ 安装 Claude Code
- ✅ 配置 API 密钥和 BASE_URL
- ✅ 配置跳过新手引导（可选）
- ✅ 验证安装结果
- ✅ 安装失败自动切换国内镜像源

**⭐ 优点：**
- 完全自动化，无需任何命令行知识
- 自动处理所有版本和配置问题
- 中文界面，操作简单直观
